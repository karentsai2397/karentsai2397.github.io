您好，我是蔡思涵 (Karen Tsai)，畢業於國立清華大學 資訊系統與應用所，主修人機互動 ( Human-Computer Interaction, HCI )。目前任職於鴻海科技集團，擔任前端介面設計師一職，負責網頁設計與開發、App介面規劃設計。融入使用者經驗設計思維，提供友善易懂的操作介面，帶給使用者良好的體驗。

主要的工作項目為：
  1. 網頁設計與開發：
    使用者介面流程設計規劃並進行整體網站視覺與Mockups設計。完稿後使用Sublime Text開發響應式網頁網頁 ( HTML5 / CSS / jQuery / Bootstrap)，具有 sass / scss 進行開發前端頁面經驗，Tortoise SVN / git 將code與圖檔做版本上的管理。主要負責前端開發，交由後端工程師進行套版與程式開發。
  2. App介面規劃設計：
    依據客戶需求進行功能規劃，由使用者的觀點出發，設計跨平台的UX使用體驗 ( 包含 Desktop Web / Mobile Web / APP )。APP規劃初期需要調查研究相關的產品及其介面設計與UI Flow，針對不同平台操作與介面設計特性進行Flow與UI設計；Flow規劃的過程中，與PM及工程師討論流程的合理性、可行性。介面設計完稿後，依ios / Android不同平台提供UI標記文件供工程師開發介面。後續，設計APP相關宣傳圖檔與網站。
